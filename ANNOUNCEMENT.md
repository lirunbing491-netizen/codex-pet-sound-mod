# Announcement Copy

## 中文

我发布了一个通用的 Codex 桌宠互动音频 Mod：Codex Pet Sound Mod。

它不是某个特定桌宠包，也不包含任何 Codex 本体文件。它会基于用户本机已经安装的 Codex 创建一个可写 overlay，并在桌宠点击、拖动、互动状态变化和本地任务开始时播放音效。

核心特点：

- 支持任意 Codex 自定义桌宠。
- 音效目录约定为 `.codex\pets\<pet-id>\sounds\*.wav`。
- 切换桌宠后会自动切换到对应桌宠的声音目录。
- 启动 modded Codex 时会自动启动音频 bridge。
- 官方 Codex 更新后可以手动重新运行安装器刷新 overlay。
- 不修改官方 Codex 安装目录，不分发 Codex 本体。

项目地址：

https://github.com/lirunbing491-netizen/codex-pet-sound-mod

## English

I released Codex Pet Sound Mod, a generic interaction sound mod for Codex custom desktop pets.

It is not tied to any specific pet and does not redistribute Codex itself. It builds a writable overlay from the Codex installation already on your PC, then plays sounds when the pet is clicked, dragged, becomes interactive, or when a local Codex task starts.

Highlights:

- Works with any Codex custom pet.
- Uses `.codex\pets\<pet-id>\sounds\*.wav` for per-pet sounds.
- Automatically follows the currently selected custom pet.
- Starts the sound bridge together with the modded Codex launcher.
- Can be rebuilt after official Codex updates.
- Does not modify the official Codex installation directory or redistribute Codex files.

Repository:

https://github.com/lirunbing491-netizen/codex-pet-sound-mod
